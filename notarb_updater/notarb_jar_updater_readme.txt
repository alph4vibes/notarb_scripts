NotArb Updater
v1.0

This script automatically checks for the latest version of the NotArb JAR file and updates it in your local environment if needed.
Instructions

    Place the Script: Put the script in the NotArb "libs" folder where your notarb_java.sh file is located.
	
    Run the Script: Execute the script.

The program will check if an update is needed. 
If the current JAR version is outdated, it will download the latest version and update the notarb_java.sh file accordingly.

Requirements
    Python 3.x
    requests library (install using pip install requests if not already installed)
